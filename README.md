# AI Recommendation System

## Description

This project is a simple AI Recommendation System developed using Python.

The program accepts a user's interests, compares them with predefined categories using similarity matching, and recommends the most relevant courses or topics based on the highest match score.

## Features

- Takes user interests as input.
- Uses simple AI recommendation logic.
- Calculates similarity using keyword matching.
- Displays personalized recommendations.

## Technologies Used

- Python 3
